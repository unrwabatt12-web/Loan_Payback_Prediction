# Loan Payback Prediction Application

# Project Description

This project is a Machine Learning–based application designed to predict whether a borrower will successfully repay a loan. The system uses historical loan data, a trained ML model, and a Flask-based backend API connected to a web frontend.

# Technologies Used

1. Python
2. Flask
3. Scikit-learn
4. Pandas
5. mysql
6. HTML, CSS, JavaScript

# Project Structure
backend/        → Flask API and ML model
frontend/       → Web user interface
dataset/        → Training dataset


# How to Run the Project
1. Install Dependencies
pip install -r backend/requirements.txt

2. Run Flask Backend
cd backend
python app.py

3. Run Frontend
Open:
frontend/index.html in a web browser.

# Features
-Loan payback prediction
-Trained ML model with scaler
-RESTful Flask API
-Web-based frontend interface
